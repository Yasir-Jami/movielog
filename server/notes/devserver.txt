Quick Setup Guide: https://docs.bitnami.com/general/infrastructure/nodejs/get-started/get-started/

[Server]
// Restart systemctl and enable newly-added server daemon
sudo systemctl daemon-reexec
sudo systemctl daemon-reload
sudo systemctl enable mserver
sudo systemctl start mserver

// Check status
sudo systemctl status mserver

File located at /etc/systemd/system/mserver.service

Alternate method using tmux (current method):

1. tmux
2. node server.cjs
3. Press Ctrl+B, D to detach
4. Server should function without ssh connection

To delete a tmux session:
1. tmux ls (check list of sessions)
2. tmux kill-session -t [session name]

To access a detached session:
tmux attach -t n (where n is session name)

[Bitnami/Apache]
Check if running apache or nginx:
1. sudo /opt/bitnami/ctlscript.sh status
Restart all Bitnami services
1. sudo /opt/bitnami/ctlscript.sh start

Renew Let's Encrypt Certificate
1. sudo /opt/bitnami/ctlscript.sh stop
2. sudo /opt/bitnami/letsencrypt/lego --tls --email="EMAIL-ADDRESS" --domains="DOMAIN" --path="/opt/bitnami/letsencrypt" renew --days 90
3. sudo /opt/bitnami/ctlscript.sh start
